#Kleine Erklärung zum WS-Discord#

[Linux Version]
--------------------------------------------------------------------------
Ihr habt Probleme mit dem Ticketsystem?
Ihr könnt keine Tickets erstellen?

-Ihr musst wenn ihr keine Tickets erstellen könnt einfach die [Message ID] vom Ticket Embed kopieren und in der Config unter [ticket-1-messageid] eintragen.

-Dies machst du dann auch mit den anderen Tickets
--------------------------------------------------------------------------

Wie installiere ich Java auf meinen Server wenn ich das noch nicht habe?

Schritt 1:
apt-get update

Schritt 2:
sudo apt install openjdk-11-jre-headless

--------------------------------------------------------------------------

Wie starte ich meinen Bot?

Schritt 1:
cd /home/und dann den Pfad wo der Bot gespeichtert ist

Schritt 2:
chmod 777 ./start.sh

Schritt 3:
./start.sh
